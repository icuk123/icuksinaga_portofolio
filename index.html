02 // SPESIALISASI

Bidang Yang Saya Minati

Internet of Things

Menghubungkan perangkat keras ke internet untuk pengumpulan data nirkabel, monitoring real-time, dan otomasi jarak jauh. Berpengalaman menerapkan sistem telemetri sensor terdistribusi.

  • Protokol: MQTT, CoAP, HTTP, LoRa
  • Platform: ThingsBoard, Blynk, AWS IoT
  • Perangkat: ESP32, ESP8266, Raspberry Pi
  • Teknik: Low-Power Sleep, Mesh Network

Embedded Systems

Merancang sirkuit elektronik khusus (PCB) dan memprogram firmware tingkat rendah yang andal. Fokus pada efisiensi daya, manajemen memori, dan komputasi real-time.

  • Platform: STM32 (ARM Cortex), AVR, ESP-IDF
  • RTOS: FreeRTOS, RT-Thread
  • Bahasa: C, C++, Bare-metal Assembly
  • Hardware: KiCad, Eagle, Logic Analyzer

Robotics

Mengintegrasikan sensor, aktuator, dan algoritma kontrol untuk memampukan mesin bergerak secara cerdas dan otonom. Tertarik pada robot navigasi darat dan lengan robotika.

  • Framework: ROS (Robot Operating System) 1/2
  • Navigasi: SLAM, LiDAR, AMCL, Obstacle Avoidance
  • Kontrol: Algoritma PID, Kinematika Balik
  • Vision: OpenCV, Pendeteksi Objek YOLO
